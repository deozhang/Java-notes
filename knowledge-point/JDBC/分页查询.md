6)分页查询
SQL语句：
select * from emp_zhang limit ?,?;
ps:
第一个问号表示每页的开始下标(从0开始)
第二个问号表示每页的记录数

//每页显示5条记录
第一页 0-4 limit 0,5
第二页 5-9 limit 5,5
第三页 10-14 limit 10,5
...
ps:
pageSize:每页的记录数
page:第几页
//计算公式
int begin=(page-1)*pageSize;

//核心代码
String sql="select * from emp_zhang limit ?,?";
ps=conn.prepareStatement(sql);
//问号赋值
ps.setInt(1,begin);
ps.setInt(2,pageSize);
//查询
rs=ps.executeQuery();
